github+hexo搭建个人博客

发表于2022-10-07|更新于2022-10-07
|字数总计:1.3k|阅读时长:4分钟|阅读量:

前言：

​ 随着互联网浪潮的翻腾，国内外涌现出越来越多优秀的社交网站让用户分享信息更加便捷。一名真正的程序猿或许可以自己动手编写一个属于自己的博客网站，然而，我只是一名半路出家的半程序猿，也想要拥有一个属于自己的博客网站，写写文章记录自己的生活、工作、学习等。

​ 花了几天时间学习基于github+hexo搭建个人博客，并安装butterfly主题，在此，写下简易搭建完博客的正式第一篇博客，记录此过程。

Hexo

​ Hexo是一款基于Node.js的静态博客框架，依赖少，易于安装使用，可以方便的生成静态网页托管在GitHub和Heroku上，是搭建博客的首选框架。同时，可以更换不同主题，自定义各种样式。

博客搭建需要的两个软件：

Node.js
git

修改配置文件用到的软件：

vscode : 适合有开发基础的程序猿，非常好用。

Hexo安装

注意：由于npm是从国外服务器下载，受网络影响较大，可能会出现异常，所以我们需要连接国内淘宝官方镜像服务器来进行：在新建的blog文件内右击鼠标—>Git Bash Here ,然后输入以下命令：

npm install -g cnpm --registry=https://registry.npm.taobao.org

如果运行上述命令后，后续使用命令cnpm开头

hexo安装此处有两种方案：（推荐第二种）

1、选一个磁盘作为博客文件存放位置，新建一个文件夹（例如：blog），创建完后，不要进入该文件夹内，在创建该文件夹处，右击鼠标—>Git Bash Here ,然后输入以下命令：

# hexo框架安装
npm install -g hexo-cli
# 安装完成hexo后，输入下面的命令 ，此处新建文件夹为blog
hexo init <新建文件夹的名称>  
cd <新建文件夹名称>
#安装博客所需的依赖文件
npm install

2、选一个磁盘作为博客文件存放位置，新建一个文件夹（例如：blog），进入该文件夹内，右击鼠标—>Git Bash Here ,然后输入以下命令：

# hexo框架安装
npm install -g hexo-cli
#上面命令运行完后，输入下面的命令
hexo init
#安装博客所需依赖文件
npm install

安装完成后，此时blog文件夹内多了许多文件，站点目录如下：

注意：后续命令均在站点目录下（即blog文件内）使用Git Bash运行。

此时，hexo本地博客框架搭建已经完成，运行如下命令：

#静态文件生成 hexo generate 简写 hexo g
hexo g
#启动本地服务器 hexo server  简写 hexo s
hexo s

说明：如果没有server，则需要安装hexo-server插件

#安装hexo-server插件
npm install hexo-server --save

运行完hexo s命令后，出现如下图：

浏览器中打开<http：//localhost:4000>,可以看到一个网页，说明hexo博客已经成功在本地运行。

博客部署到Github Pages

上述步骤，本地博客成功搭建，接下来将本地博客发布到github pages，然后可以共享至其他人观看。

首先需要安装发布的插件，在站点目录下执行下面的命令：

npm install hexo-deployer-git --save

将本地目录与github关联起来，输入下面命令：

ssh-keygen -t rsa -C "你的注册邮箱"

四个回车键之后完成密钥生成，该密钥文件在如下图位置：

然后在C:/Users/[username]目录下找到名为.ssh的文件夹， 文件夹内会有两个文件，一个id_rsa.pub一个id_rsa，用文本编辑器打开id_rsa.pub，复制里面的的内容。然后打开Github，点击右上角的头像Settings 选择SSH and GPG keys。点击New SSH key将之前复制的内容粘帖到Key的框中。上面的Title可以随意，点击Add SSH key完成添加。

回到bash 界面，测试是否与github链接成功，输入如下命令：

ssh -T git@github.com

点击回车，然后会出现一个询问内容，输入yes，回车，会出现一段内容，**Hi ! You’ve successfully authenticated, but GitHub doesnot provide shell access.。说明连接成功。此处这个**应该是你Github的用户名。

github新建一个仓库

注意：仓库名格式为：用户名.github.io，此名即为后续博客域名，通过https://用户名.github.io即可访问该博客。

使用vscode打开站点目录下_config.yml文件，拖到文件末尾，修改deploy属性：

此时需要安装一个deploy插件，输入命令：

cnpm install hexo-deployer-git
博客部署

将本地博客部署到github pages，输入命令：

#将本地文件部署到github 
hexo d

至此，整个博客建站，部署等已经完成，此时回到浏览器输入：https://用户名.github.io，即可访问你的博客。
